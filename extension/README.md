# Aria Assist Chrome Extension

Aria Assist is a companion Chrome Extension to the Aria ADA Compliance Auditor. It acts as an immediate accessibility patch for users, providing tools that can be injected into any non-compliant website.

## Features Included
1. **Line Reader**: A focused green reading bar that tracks your mouse.
2. **Text-to-Speech**: Simply highlight any text on the page, and the browser will read it aloud to you.
3. **High Contrast**: Instantly bumps the contrast and dims the brightness of any webpage.
4. **Readable Font**: Overrides all fonts on the page to a highly legible, dyslexic-friendly font with increased letter spacing.

## How to Install it in Chrome (Takes 30 seconds)

1. Open a new tab in Google Chrome and type `chrome://extensions/` into the URL bar and hit enter.
2. Look at the top right corner of the page and turn **ON** the toggle switch for **"Developer mode"**.
3. In the top left corner, click the **"Load unpacked"** button.
4. A file browser will open. Navigate to your project folder (`c:\Meera Files\Antigravity\TSA Agent 3`) and select the **`extension`** folder. Click "Select Folder".

That's it! You will now see "Aria Assist" in your extensions list.

## How to Demo It

1. Click the puzzle piece icon 🧩 in the top right of your Chrome browser (next to your profile picture) and "Pin" the Aria Assist extension so the icon is always visible.
2. Open *any* website (like Wikipedia, or a mock non-compliant website).
3. Click the Aria Assist icon. You will see a beautiful popup that perfectly matches your Aria web app's color theme.
4. Toggle any of the switches and watch the website instantly become more accessible!

**To demo Text-to-Speech:** Turn the toggle ON, then use your mouse to highlight a paragraph of text. As soon as you let go of your mouse click, it will read it aloud!
